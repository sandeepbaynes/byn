# @byn/cli

This package **reserves the `@byn` scope** on npm for **byn** — a local-first
secure secrets vault and credential manager. It is a **Go CLI**, not an npm
package, and this placeholder **installs no executable**.

Install the real tool:

```sh
go install github.com/sandeepbaynes/byn/cmd/byn@latest
```
